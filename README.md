# Superstore-Sales-Profit-Dashboard ( Interactive Dashboard Created using MS Excel)
## Project Objective
To analyze Superstore sales and profitability data and present key business insights through an interactive Excel dashboard that supports data-driven decision-making.

## Dataset Used
-<a href="https://github.com/Ayush0713/superstore-sales-profit-excel-dashboard/blob/main/Superstore%20Sales%20%26%20Profit%20Dashboard.xlsx">Dataset</a>

## Questions(KPIs)
- identify sales and profit trends over time
- Compare category and sub-category performance
- Highlight loss-making products and regions
- Track key KPIs such as Total Sales, Total Profit, Orders, and Profit Margin
- Enable interactive analysis using slicers and filters
